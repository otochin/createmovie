# 変更履歴

このファイルには、プロジェクトの**重要な変更の要約**を記録します。  
**利用者・リリースノート向け**に「何が追加・変更・修正されたか」をカテゴリ別に簡潔に書きます。

- **DEVELOPMENT_LOG.md との使い分け**: 詳細な実施内容・変更ファイル一覧・完了チェックリストは `docs/DEVELOPMENT_LOG.md` に記載します。CHANGELOG は要約のみとし、バージョンタグ付け時に [Unreleased] を [バージョン] に移します。

形式は [Keep a Changelog](https://keepachangelog.com/ja/1.0.0/) に基づいています。

## [Unreleased]

### Added
- サムネイル用テキスト案のGPT 3案生成と表示・コピー（台本ページ、Canva用）
- 参考台本から「核心部分」を抽出し、台本生成の後半で反映する機能
- 「参考台本核心部」入力欄（核心抽出の手がかりとして利用）
- 画像生成画面で「画像を指定」によりPCから画像を選んで個別シーンにセットする機能
- 動画編集機能の実装
  - MoviePyを使用した動画生成
  - 画像と音声の同期処理
  - 字幕の自動追加機能（PILベース）
  - 日本語フォント対応（ヒラギノ角ゴシック）
  - 字幕スタイル設定機能
  - 動画プレビュー表示（30%サイズ）
  - 動画エクスポート・ダウンロード機能
- プロジェクト初期化
- システム定義書（README_PLAN.md）の作成
- ディレクトリ構成案の策定
- requirements.txtの作成
- README.mdの作成
- .gitignoreの設定
- 開発ログ（DEVELOPMENT_LOG.md）の作成
- 変更履歴（CHANGELOG.md）の作成

### Changed
- 台本生成のデフォルト指示に冒頭フックの一文を追加
- dialogue_for_tts に句読点をこまめに入れるプロンプト指示を追加
- 1シーンあたりの文字量を約2倍に（12〜16文字/秒）、デフォルト値を総時間180秒・シーン数18・スタイル教育に変更
- ストック画像紐づけで前回との重複チェックを廃止（今回の紐づけ内のみ重複なし、何度でも押せる）
- 個別指定PNGのリサイズで透過を保持（背景動画が透けて見える）
- 画像マッピングの保存・読み込みで絶対パス化と resolve による正規化
- 画像を指定のアップロードを収集して一括処理するよう変更（2件目以降が画面に反映される）
- 動画表示サイズを30%に縮小（画面からはみ出る問題を解決）
- MoviePy 1.0.3にダウングレード（互換性のため）
- 字幕機能をTextClipからPILベースの実装に変更

### Fixed
- 画像を指定時の tempfile 参照エラー（参考画像ブロック内の重複 import を削除）
- 個別指定したPNGが黒背景になる問題（透過保持のリサイズに変更）
- 画像を指定で2件目以降がブラウザに反映されない問題（一括処理化とマッピング同期）
- Pillow 11.3.0との互換性問題（Image.ANTIALIAS → Image.LANCZOS）
- 日本語字幕の文字化け問題（ヒラギノ角ゴシックフォントを使用）
- 字幕クリップがNoneの場合のエラーハンドリング

## 変更履歴の記録形式

### [バージョン] - YYYY-MM-DD

#### Added（追加）
- 新機能の追加

#### Changed（変更）
- 既存機能の変更

#### Deprecated（非推奨）
- まもなく削除される機能

#### Removed（削除）
- 削除された機能

#### Fixed（修正）
- バグ修正

#### Security（セキュリティ）
- セキュリティ関連の修正

## バージョン管理

- **メジャーバージョン**: 大きな機能追加や破壊的変更
- **マイナーバージョン**: 新機能追加（後方互換性あり）
- **パッチバージョン**: バグ修正や小さな改善

例: v1.0.0 → v1.1.0 → v1.1.1 → v2.0.0
